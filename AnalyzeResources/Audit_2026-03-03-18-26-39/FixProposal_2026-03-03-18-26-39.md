# 资源修复建议 (Fix Proposal)

根据资源审计报告 `ResourceAudit_2026-03-03-18-26-39.md` 及最佳实践指南，提出以下修复建议。

## 1. 纹理优化 (Texture Optimization)

**涉及资源**: `Assets/SimpleNaturePack/Textures/NaturePackLite_Texture_01.png`

*   **问题 1**: Mipmaps 未开启。
    *   **建议**: 开启 `Generate Mip Maps`。
    *   **理由**: 对于非 UI 的 3D 物体纹理，开启 Mipmaps 可减少远处噪点并提高 GPU 缓存命中率，节省带宽。
    *   **参考**: [Texture Guidelines](references/texture_guidelines.md#2-mipmaps-多级渐远纹理)

*   **问题 2**: 缺少特定平台覆盖设置 (Android/iOS)。
    *   **建议**: 为 Android 和 iOS 平台启用 `Override for ...`，并设置压缩格式为 `ASTC 6x6`。
    *   **理由**: 默认设置可能导致包体过大或显存占用过高。ASTC 格式在保持画质的同时提供优秀的压缩率。
    *   **参考**: [Texture Guidelines](references/texture_guidelines.md#4-compression-format-压缩格式)

## 2. 模型优化 (Model Optimization)

**涉及资源**: `Assets/SimpleNaturePack/Models/*.fbx` (共 24 个模型)

*   **问题 1**: 网格压缩 (Mesh Compression) 未开启。
    *   **建议**: 将 `Mesh Compression` 设置为 `Low` 或 `Medium`。
    *   **理由**: 压缩网格数据可显著减小 Asset Bundle 包体大小，且通常对运行时性能无负面影响。
    *   **参考**: [Model Guidelines](references/model_guidelines.md#2-mesh-compression-网格压缩)

*   **问题 2**: 网格优化 (Optimize Mesh) 未开启。
    *   **建议**: 勾选 `Optimize Mesh` 选项。
    *   **理由**: Unity 会重新排列顶点和索引以优化 GPU 缓存命中率，从而提高渲染性能。
    *   **参考**: [Model Guidelines](references/model_guidelines.md#3-optimize-mesh-网格优化)
